## Design Principles

1. **Mobile-First**: All screens designed for mobile first, progressively enhanced for tablet and desktop
2. **Warm Indian Aesthetic**: Gold + Burgundy color scheme reflects traditional Indian marketplaces
3. **Mode Clarity**: Three interaction modes visually distinct with clear color coding
4. **Accessibility**: WCAG 2.1 AA minimum contrast ratios, proper focus states, semantic HTML
5. **Consistency**: Single design system across web (Next.js) and mobile (Expo React Native)
6. **Help Embedded**: Every screen contains contextual help strings for user assistance
